# Wizard's Elixirs

A unique, fun, fully-polished Minecraft **Bedrock Edition** addon designed to
work on iPad (and any other Bedrock platform: iPhone, Android, Windows 10/11,
Xbox, PlayStation, Switch).

The pack adds **five hand-crafted magical elixirs** with unique effects, pixel-art
textures, crafting recipes, and themed lore.

| Elixir          | Effects                                     | Main ingredient |
| --------------- | ------------------------------------------- | --------------- |
| Flame Brew      | Fire Resistance II, Strength II             | Blaze Powder    |
| Frost Draught   | Slow Falling, Resistance II, Water Breathing| Ice             |
| Storm Tonic     | Speed III, Jump Boost III                   | Feather         |
| Nature's Sip    | Regeneration II, Absorption II, +6 Hunger   | Melon Slice     |
| Void Essence    | Night Vision, Invisibility                  | Ender Pearl     |

Each recipe uses the classic potion base (glass bottles + nether wart) in a
crafting table, with the signature ingredient below. One craft yields **two**
elixirs.

```
 B       B = Glass Bottle
NBN      N = Nether Wart
 X       X = signature ingredient (see table)
```

## Install on iPad (one tap)

1. Share the file `dist/WizardsElixirs.mcaddon` to your iPad (AirDrop,
   iMessage, email, Google Drive, Dropbox, Files app...).
2. Tap the file. iPadOS will ask which app should open it — choose
   **Minecraft**.
3. Minecraft launches and imports both the behavior and resource packs
   automatically. You'll see a confirmation at the top of the screen.
4. Create or edit a world. In world settings enable **both**:
   - `Behavior Packs` → `Wizard's Elixirs - Behavior`
   - `Resource Packs` → `Wizard's Elixirs - Resources`
5. Enter the world and craft elixirs at a crafting table.

If tapping the `.mcaddon` doesn't give you a "Open in Minecraft" option,
rename the file to `WizardsElixirs.mcaddon` (case matters) or try opening it
from the Files app → Share → Minecraft.

## Install on Android / Windows / Console

Same flow: open the `.mcaddon` file and Minecraft will import it. On Windows
you can double-click the file from File Explorer.

## Developer notes

- Format version: `1.20.50` (works on Minecraft Bedrock 1.20.50 and newer).
- Namespace: `welix:`
- Pack structure is a clean two-pack addon (Behavior Pack + Resource Pack)
  linked via manifest dependencies.
- Custom items use the stable `minecraft:food` component with baked-in
  `effects`, so no scripts are required and the pack is safe for Realms and
  multiplayer.
- Textures are 16x16 hand-placed pixel art generated by
  `src/make_textures.py` (re-runnable).
- The `.mcaddon` file is just a zip — the `.zip` copy in `dist/` has
  identical contents if you prefer to extract manually into `com.mojang/`.

## Rebuild from source

```sh
python3 src/make_textures.py
cd build
zip -r -9 ../dist/WizardsElixirs.mcaddon WizardsElixirs_BP WizardsElixirs_RP
```
