OmniSight API Reference

Unified prediction market data infrastructure. REST API + WebSocket streaming.

https://api.omnisight.dev/v1

Authentication

Include your API key in the X-API-Key header. Free tier endpoints work without authentication (rate-limited).

curl -H "X-API-Key: sk-your-key" https://api.omnisight.dev/v1/markets

Rate Limits

Rate limits are enforced per API key. Response headers include current usage:

X-RateLimit-Limit: 600
X-RateLimit-Remaining: 594
X-RateLimit-Reset: 1712688000

List Markets FREE

GET /v1/markets

List markets across all platforms or filter by platform, category, and status.

ParameterTypeDescription
platformstringFilter by platform (polymarket, kalshi, pinnacle, betfair)
categorystringFilter by category (politics, crypto, sports, economics)
statusstringMarket status: active, closed, resolved (default: active)
limitintegerMax results (default: 50, max: 500)
offsetintegerPagination offset (default: 0)

Response

[
  {
    "id": "poly-abc123",
    "platform": "polymarket",
    "title": "US Presidential Election 2028",
    "yes_price": 0.42,
    "no_price": 0.58,
    "volume_24h": 18420000,
    "liquidity": 5200000,
    "status": "active"
  }
]

Order Book PRO

GET /v1/orderbook/{market_id} PRO

Real-time order book snapshot with bid/ask depth, imbalance, and spread analytics.

ParameterTypeDescription
market_idrequiredstringMarket identifier (path param)
platformrequiredstringPlatform name

Response

{
  "market_id": "abc123",
  "bids": [{"price": 0.64, "size": 5200}, ...],
  "asks": [{"price": 0.66, "size": 4800}, ...],
  "mid_price": 0.65,
  "spread": 0.02,
  "imbalance": 0.12
}

Whale Alerts PRO

GET /v1/whales/alerts PRO

Recent whale trade alerts across all platforms. Filter by market, platform, or minimum USD value.

ParameterTypeDescription
market_idstringFilter by market ID
platformstringFilter by platform
min_usdnumberMinimum USD value (default: 10000)
hoursintegerLookback window in hours (default: 24)

Arbitrage Opportunities PRO

GET /v1/arbitrage PRO

Scan for cross-platform arbitrage opportunities, sorted by estimated profit.

ParameterTypeDescription
categorystringFilter by category
min_profit_bpsnumberMinimum profit in basis points (default: 10)

Market Microstructure PRO

GET /v1/microstructure/{market_id} PRO

Full microstructure analytics: spread, depth, fill rates, slippage, VWAP, whale flow.

WebSocket: Price Stream PRO

WS /v1/ws/prices PRO

Real-time price streaming. Send subscription message, receive continuous updates.

Subscribe

{"subscribe": ["market-id-1", "market-id-2"], "platform": "polymarket"}

Message Format

{
  "market_id": "abc123",
  "platform": "polymarket",
  "yes_price": 0.6523,
  "no_price": 0.3477,
  "timestamp": "2026-04-09T14:23:00Z"
}

WebSocket: Whale Alert Stream PRO

WS /v1/ws/whales PRO

Real-time whale trade alerts. No subscription needed — all alerts are streamed automatically.

WebSocket: Arbitrage Stream PRO

WS /v1/ws/arbitrage PRO

Real-time arbitrage opportunity streaming as they are detected across platforms.
