# Dreamhouse Deluxe

The ultimate build-a-home kit for **Minecraft Bedrock Edition** (iPad, iPhone,
Android, Windows 10/11, Xbox, PlayStation, Switch).

Built for a young builder who loves making unique houses. One-tap install on iPad.

---

## What's inside

### 45 Building Blocks — 3 themed sets

| Theme            | Blocks | Vibe                                               |
| ---------------- | ------ | -------------------------------------------------- |
| **Fairy Cottage** | 15     | Mushroom caps, flower walls, vine ropes, glow flowers, fairy glass |
| **Candy Castle**  | 15     | Gumdrop blocks, peppermint walls, frosting, lollipop spirals, gingerbread |
| **Cozy Modern**   | 15     | Pastel planks (6 colors), rainbow glass, carpet tiles, star & heart wallpaper |

### 12 Furniture Blocks with real 3D geometry

Beds (pink & blue) · Couches (pink & teal) · Armchair · Dining table & chair ·
Coffee table · TV (glows!) · Refrigerator · Bookshelf · Glowing lamp

Every piece is a custom Bedrock block with its own 3D model — not a flat
texture hack.

### 5 Magic Instant-Build Wands (Script API)

| Wand             | What it builds                        | Cooldown |
| ---------------- | ------------------------------------- | -------- |
| Fairy Cottage    | 7×9 mushroom cottage with bed & lamp  | 8 sec    |
| Candy Castle     | 9×11 candy castle with towers         | 8 sec    |
| Cozy Cabin       | 7×8 pastel cabin with chimney         | 8 sec    |
| Treehouse        | Giant tree with a platform house      | 8 sec    |
| Rainbow Tent     | Colorful A-frame tent                 | 8 sec    |

Tap any block surface with a wand — the structure appears instantly with
sparkle particles and a level-up sound.

### 2 Companion Pets

- **Puppy** — tame with bones or cooked beef. Follows you, sits on command,
  tail wags. Immune to fall damage.
- **Bunny** — tame with carrots. Hops around, wiggles ears, follows you.

Both have full 3D models with walk, idle, and sit animations.

### 62 Crafting Recipes

Every block and wand is craftable at a crafting table using vanilla ingredients.
Building blocks yield 8 per craft. Wands use an amethyst shard + stick +
themed ingredient.

### Quick-start chat commands

| Command  | Effect                              |
| -------- | ----------------------------------- |
| `!wands` | Gives you all 5 magic wands         |
| `!pets`  | Spawns a puppy and bunny beside you |

## Install on iPad (one tap)

1. Share `dist/DreamhouseDeluxe.mcaddon` to your iPad (AirDrop, iMessage,
   Files, email, etc.).
2. Tap the file → choose **Minecraft**.
3. Minecraft imports both packs automatically.
4. Create or edit a world → enable **both** packs in world settings:
   - Behavior Packs → **Dreamhouse Deluxe**
   - Resource Packs → **Dreamhouse Deluxe**
5. **Important:** Under world settings → Experiments, enable
   **Holiday Creator Features** (needed for custom blocks) and
   **Beta APIs** (needed for Script API wands).
6. Enter the world. Type `!wands` in chat to get started.

## Technical details

- **Format:** Bedrock 1.21.0+ (format_version 1.20.50 for items/blocks)
- **Script API:** `@minecraft/server` v1.14.0
- **Namespace:** `dh:`
- **Total assets:** 247 files (57 blocks, 12 geometry models, 5 items,
  2 entities, 62 recipes, 64 textures, 5 animations, 1 JS script)
- **No external dependencies** — self-contained addon, safe for Realms
- **Textures:** 16×16 hand-placed pixel art, generated by Python scripts
  in `src/` (re-runnable)

## Rebuild from source

```sh
python3 src/generate_blocks.py
python3 src/generate_furniture.py
python3 src/generate_wands.py
python3 src/generate_pets.py
cd build
zip -r -9 ../dist/DreamhouseDeluxe.mcaddon DreamhouseDeluxe_BP DreamhouseDeluxe_RP
```
